Vector Direction Drawing Game

Practice translating direction phrases into arrows. Draw an arrow with the correct direction; the length does not matter.

How to use

  1. Read the target direction, such as “40° north of west.”
  2. Draw an arrow from the origin by clicking or dragging in the graph area.
  3. The live angle readout uses the last main axis your arrow touched while drawing as the reference axis.
  4. Choose “Check direction” to see whether the direction of your vector exactly matches the target direction.
  5. The length of your arrow doesn't matter, but the direction of very short arrows is unclear and are not checked as a result.

Keyboard users can use the angle and length controls, or focus the graph and use arrow keys to rotate the arrow.

Direction convention

“North of east” means start facing east and turn toward north.
“East of north” means start facing north and turn toward east.
Those are different directions, even if the same number of degrees appears.

While drawing, the angle readout uses the last axis your arrow touched as the reference axis.

This uses standard math angles: 0° is east/right, 90° is north/up, 180° is west/left, and 270° is south/down. When the slider passes through one of these axes, that axis becomes the reference axis.
