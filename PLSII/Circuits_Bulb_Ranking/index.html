Circuit Brightness Ranking Game

Rank identical bulbs from brightest to dimmest in randomly generated series-parallel circuits. The game checks the voltage across each bulb internally, but the puzzle is meant to be solved conceptually.

How to use

Look at the circuit and decide which bulb or bulbs are brightest. Click all bulbs that belong in the brightest group. Then choose Next dimmer group and click the next set of equal-brightness bulbs. Continue until every bulb is placed.

Bulbs in the same group are treated as equally bright. For example, A = D > B = C means A and D are tied for brightest, while B and C are tied for dimmer.

Keyboard option: use the bulb buttons in the controls panel, or use the listed keyboard shortcuts.

Controls

Build your ranking

Current brightness group: 1

Click a bulb once to add it to the current brightness group. Selected bulbs are marked in the center of the bulb on the diagram.

Game actions

Keyboard shortcuts

  • A–H: add that bulb, if present
  • Space: next dimmer group
  • Backspace: previous group
  • Enter: check answer
  • R: reset ranking
  • N: new circuit
